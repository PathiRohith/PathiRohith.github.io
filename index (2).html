Hybrid RAG · Clinical Data Featured

TrialMind

An intelligent question-answering system built on ClinicalTrials.gov data. TrialMind fuses dense vector search with BM25 sparse retrieval via reciprocal rank fusion, reranks candidates with a cross-encoder model, and generates grounded answers with a local LLM. Designed to run fully offline using open-source HuggingFace models — no external API keys required.

Multi-Agent · LangGraph

SalesOrbit

A multi-agent AI system for B2B sales teams. Ingests prospect signals (LinkedIn roles, intent data), retrieves context from a dual-RAG knowledge base spanning text files and OCR-scanned business cards, scores each prospect with a rule-based, signal-aware engine, and recommends the best next action — orchestrated as a LangGraph state machine with a FastAPI backend for event-driven ingestion and a Streamlit UI for manual runs.

Human-in-the-Loop · Compliance

ExpenseIQ

Automates the review of employee expense receipts against company spending policy. Extracts text from PDFs, images, or text files, retrieves the most relevant policy sections from a vector knowledge base, and asks Gemini for a structured verdict — Compliant, Flagged, or Rejected — with confidence score, reasoning, and a cited policy quote. Managers can override AI verdicts with comments, a full audit trail lives in SQLite, and a built-in Policy Chat answers plain-English policy questions.
